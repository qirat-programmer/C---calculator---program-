# Basic Calculator Program in C

This project is a simple **Calculator Program written in C language**.  
It performs basic arithmetic operations using the **switch case statement**.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- User input based calculations

## How It Works
1. The user enters two numbers.
2. The program asks the user to select an operation.
3. The operation is performed using a **switch case** statement.
4. The result is displayed on the screen.

## Technologies Used
- C Programming Language
- Dev-C++

This project was created as part of a **programming assignment** to practice:
- Conditional statements
- Switch case
- User input in C
